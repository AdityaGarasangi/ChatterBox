About ChatterBox

ChatterBox is a real-time chat application designed to connect people through engaging conversations. Whether you want to discuss topics of interest, share ideas, or simply chat with friends, ChatterBox provides a user-friendly interface to facilitate these interactions.

Features

  • Real-time messaging with WebSocket integration
  • Multiple chat rooms to join and engage
  • User-friendly interface with a modern design
  • Responsive layout for mobile and desktop users
  • Username display for easy identification

Technologies Used

ChatterBox is built using the following technologies:

  • Django for the backend framework
  • HTML, CSS, and Tailwind CSS for styling
  • JavaScript for frontend interactivity
  • WebSockets for real-time communication
